# Ciclos
- Los ciclos permiten repetir un conjunto de declaraciones, un número controlado de veces.
- Recomendación: Cuando utilices while, mantén la inicialización de la variable muy cerca de la condición del cilco, con suerte que sea la expresión justo antes."Carlos Obregón"
- Hay dos formas que son muy comunes: Ajuste de sumando 1 o restando 1, pero vienen en dos formas. `++n | n++`
- La diperencia entre un post-incremento y un post-decremento, pre-incremento y pre-decremento solo es importante cuando los usas en la misma linea con otros operadores."Carlos Obregón"
- `Recordatorio` Un String es una secuencia de caracteres, cada carácter esta asociado con un índice. podemos usar metodos como `charAt()` para acceder al carácter actual que se está atravesando.

- Cuándo usar ciclos for y cuándo usar ciclos While? 
 1. Lo primero que es importante decir que es que no hay ninguna diferencia en funcionalidad.
 2. Todo código que utilice un ciclo for se puede reescribir como un código que use un ciclo while y viceversa.
